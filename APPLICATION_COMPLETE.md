## 🚀 How to Run the Application

### Quick Start (5 Minutes)

**Step 1: Install Dependencies**
```powershell
cd backend
npm install

cd ../frontend
npm install
```

**Step 2: Start Application**

**Option A - Automatic (Recommended)**
```powershell
# Windows
.\start.bat

# Mac/Linux
chmod +x start.sh
./start.sh
```

**Option B - Manual**
```powershell
# Terminal 1 - Backend
cd backend
npm start

# Terminal 2 - Frontend
cd frontend
npm run dev
```

**Step 3: Open Browser**
```
http://localhost:5173
```
